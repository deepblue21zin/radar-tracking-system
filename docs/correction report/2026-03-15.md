### 1. `kalman_tracker.py` FilterPy 행렬 타입 오류 수정
- 기존 상태:
  - tracker가 `FilterPy`에 넘기는 `x`, `F`, `H`, `Q`, 측정값 `z`를 Python `list` 형태로 구성하고 있었다.
  - 첫 프레임에서는 track 생성까지만 진행되어 보이지 않다가, 다음 predict 단계에서 `F.T`를 쓰는 순간 `'list' object has no attribute 'T'` 에러가 발생했다.
- 수정 내용:
  - `numpy.ndarray` 기반으로 state / transition / observation / covariance 행렬을 모두 구성하도록 수정했다.
  - `Q`는 `np.block`으로 4x4 process noise 행렬을 만들도록 바꿨다.
  - predict 단계의 `F` 갱신도 ndarray 인덱싱 방식으로 맞췄다.
  - update 측정값 `z`도 ndarray로 바꿨다.
- 수정 이유:
  - FilterPy는 내부에서 `.T`와 행렬 곱을 사용하므로 list가 아니라 ndarray가 필요하다.
  - 이번 실기기 테스트에서 실제로 터진 에러를 재현 기반으로 수정한 것이다.
