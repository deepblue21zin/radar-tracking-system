REQ Package

Radar SW Requirements Package

이 패키지는 레이더 SW를 알고리즘 실험 프로젝트가 아니라, 양산형 자동차 임베디드 SW 후보 아키텍처로 설명하기 위한 요구사항 묶음입니다. 목적은 RTOS 구조, 성능 수치, 입출력 계약, fail-safe 정책을 같은 프레임 안에서 읽히게 만드는 것입니다.

Package Goals

RTOS / Task

task period, priority, queue depth, WCET, bounded buffer를 명시적으로 문서화합니다.

Performance

avg뿐 아니라 max/P95 latency, FPS, CPU, memory를 같이 비교할 수 있게 합니다.

Interface

radar input, tracking output, health bit, control packet을 계약 관점으로 정리합니다.

Safety

stale/malformed/overrun/dropped에서 무엇을 안전 상태로 볼지 정책화합니다.
