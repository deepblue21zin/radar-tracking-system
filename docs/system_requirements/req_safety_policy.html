Recommended Health Gate

frame age

`frame_age_ms > 100`이면 RESUME 금지

parser burst

`parse_failures_delta` 또는 `resync_rate`가 threshold 초과 시 unhealthy

dropped burst

recent dropped frame rate가 높으면 SLOW 또는 ALARM

overrun

buffer/queue overrun 발생 시 latest-only + control inhibit

Recovery Rule

  • stale recovery: healthy frame 3개 연속 후 clear
  • malformed burst recovery: cooldown window 후 clear
  • dropped recovery: recent window 정상화 후 clear
  • overrun recovery: utilization 정상화 후 clear
